Необходимо союлюсти:
1. Генератор и папка-исходник должны находиться в одной папке (директории).
2. В названиях папок и файлов не должно быть русских букв (кириллицы).

Принцип работы:
1. Запускаем файл SS_gen
2. По запросу указываем название папки с файлами-образцами
3. Ждём
4. В папке с индексом "-generat" готовые файлы, скидываем в нужное место.
